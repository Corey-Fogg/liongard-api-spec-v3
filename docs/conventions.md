# Conventions

- Base path: `/v3`
- camelCase field names
- Auth:
  - `Authorization: Bearer <token>`
  - `X-API-Key: <key>` where specified (ingestion-like endpoints)
- Pagination:
  - Query params: `limit`, `offset`
  - Response body: `pagination { limit, offset, count }`
- Filtering:
  - Query param: `where` (field-based expression)
  - Same pattern is used inside webhook subscriptions
- Expand:
  - Query param: `expand` (array of strings)
  - No expand-all
  - Discover supported expansions via:
    - `GET /v3/meta/objects`
    - `GET /v3/meta/assets/types`
  - `strictExpand=true` returns 422 for unsupported expand values
- Scoping:
  - Inspectors are environment-scoped
  - Dataprints are always nested under environment + inspector
  - Alerts, Assets, Identities, Environments are global collections
